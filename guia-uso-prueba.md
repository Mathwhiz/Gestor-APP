# Guia Simple Para Probar La App

Esta guia esta pensada para usar la app por primera vez, sin tecnicismos y sin tocar nada raro.

## 1. Para entrar

1. Abrir la pagina de la app.
2. Tocar o hacer click en `Continuar con Google`.
3. Entrar con el Gmail habilitado.
4. Esperar a que abra el `Dashboard`.

Si vuelve a la pantalla de login, probar una vez mas. Si sigue igual, avisar que no dejo entrar.

## 2. Que mirar primero

Cuando entra, lo mejor es seguir este orden:

1. `Dashboard`
   Ahi se ve rapido si hay tramites observados, documentos pendientes, tareas y caja.
2. `Tramites`
   Ahi estan las carpetas de trabajo.
3. `Finanzas`
   Ahi estan los cobros, gastos y la caja.
4. `Ayudas`
   Ahi estan las guias para recordar requisitos o pasos.

## 3. Para que sirve cada parte

### Dashboard

Sirve para mirar el dia de un golpe de vista.

Ahi va a ver:

- tramites urgentes
- documentos pendientes
- tareas
- caja operativa
- operaciones abiertas

### Tramites

Sirve para seguir cada gestion.

En cada tramite se puede:

- ver el cliente
- ver el vehiculo
- ver el estado
- marcar requisitos como recibidos o pendientes
- agregar notas
- cambiar el estado
- cargar un movimiento relacionado
- ver la ayuda del tramite

### Contactos

Sirve para guardar clientes, terceros, registros y proveedores.

### Vehiculos

Sirve para guardar autos o motos vinculados a tramites o a operaciones.

### Tareas

Sirve para anotar pendientes del dia o de la semana.

### Finanzas

Sirve para cargar:

- cobros
- gastos
- movimientos de caja

Tambien sirve para ver:

- cuanto entro
- cuanto salio
- cuanto queda
- en que area se mueve la plata
- que tramites siguen sin ingreso cargado

### Ayudas

Sirve para consultar guias practicas:

- que documentos pedir
- que pasos revisar
- que errores evitar
- links utiles

## 4. Como probarla sin miedo

La forma mas simple de probarla es esta:

1. Entrar al `Dashboard`.
2. Abrir `Tramites`.
3. Elegir un tramite ya cargado.
4. Marcar un requisito como recibido.
5. Agregar una nota corta.
6. Cambiar el estado.
7. Ir a `Finanzas`.
8. Cargar un cobro o un gasto chico de prueba.
9. Volver al dashboard y ver si cambian los numeros.

## 5. Que conviene cargar de verdad

Puede empezar a usarla para:

- clientes
- tramites
- tareas
- vehiculos
- cobros y gastos
- notas internas

## 6. Que no conviene esperar todavia

Todavia no esta pensada como sistema contable final ni como archivo documental completo.

Hoy sirve muy bien para:

- ordenar el trabajo
- no olvidar pendientes
- seguir tramites
- ver caja del dia
- detectar que falta cobrar

## 7. Regla simple de uso

Si tiene duda, usar esta idea:

- `Tramites` para el trabajo
- `Finanzas` para la plata
- `Tareas` para lo que falta hacer
- `Ayudas` para recordar como seguir

## 8. Si se pierde

Siempre puede volver al `Dashboard`.

Desde ahi puede empezar de nuevo y mirar:

- que esta urgente
- que esta observado
- que falta cobrar
- que tarea sigue pendiente
