Guia Simple

Como entrar, mirar y probar la app sin complicarse

Esta guia esta pensada para una primera prueba tranquila. La idea no es aprender todo de una, sino entrar, ubicar las partes principales y hacer una prueba simple sin miedo.

1. Para entrar

  1. Abrir la pagina de la app.
  2. Tocar o hacer click en Continuar con Google.
  3. Entrar con el Gmail habilitado.
  4. Esperar a que abra el Dashboard.

Si vuelve a la pantalla de login, probar una vez mas. Si sigue igual, avisar que no dejo entrar.

2. Que mirar primero

  1. Dashboard: para ver rapido como esta el dia.
  2. Tramites: para ver las carpetas de trabajo.
  3. Finanzas: para ver cobros, gastos y caja.
  4. Ayudas: para recordar requisitos y pasos.

3. Para que sirve cada parte

Dashboard

  • tramites urgentes
  • documentos pendientes
  • tareas
  • caja operativa
  • operaciones abiertas

Tramites

  • ver cliente y vehiculo
  • ver y cambiar estado
  • marcar requisitos
  • agregar notas
  • cargar movimientos
  • abrir la ayuda del tramite

Contactos

Sirve para guardar clientes, terceros, registros y proveedores.

Vehiculos

Sirve para guardar autos o motos vinculados a tramites u operaciones.

Tareas

Sirve para anotar pendientes del dia o de la semana.

Ayudas

  • que documentos pedir
  • que pasos revisar
  • que errores evitar
  • links utiles

4. Finanzas, explicado simple

Para cargar

  • cobros
  • gastos
  • movimientos de caja

Para mirar

  • cuanto entro
  • cuanto salio
  • cuanto queda
  • en que area se mueve la plata
  • que tramites siguen sin ingreso cargado

5. Como probarla sin miedo

  1. Entrar al Dashboard.
  2. Abrir Tramites.
  3. Elegir un tramite ya cargado.
  4. Marcar un requisito como recibido.
  5. Agregar una nota corta.
  6. Cambiar el estado.
  7. Ir a Finanzas.
  8. Cargar un cobro o un gasto chico de prueba.
  9. Volver al dashboard y ver si cambian los numeros.

6. Que conviene cargar de verdad

Ya conviene usarla para

  • clientes
  • tramites
  • tareas
  • vehiculos
  • cobros y gastos
  • notas internas

No conviene esperar todavia

  • contabilidad final cerrada
  • archivo documental completo
  • reporte contable profesional

7. Si se pierde

Siempre puede volver al Dashboard.

Desde ahi puede mirar de nuevo:

  • que esta urgente
  • que esta observado
  • que falta cobrar
  • que tarea sigue pendiente
